# Roadmap

## Phase 1: Data Foundation

- design calibration sheet
- collect raw ragi images
- record measured moisture labels
- store metadata
- block screenshots from dataset

## Phase 2: CV Baseline

- grid detection
- normalization
- grain segmentation
- feature extraction
- rule-based risk baseline

## Phase 3: ML Model

- ragi-specific hybrid model
- flash/no-flash fusion
- three-view aggregation
- confidence scoring

## Phase 4: Calibration and Evaluation

- crop-specific calibration
- evaluate on unseen farms/devices
- report MAE, RMSE, F1, calibration error
- tune false-safe rate

## Phase 5: App and Backend

- guided capture app
- inference API
- report generation
- meter-confirmation feedback loop

## Phase 6: Multi-Crop Expansion

- add new crop datasets
- build crop-specific calibration heads
- compare generic vs crop-specific performance
